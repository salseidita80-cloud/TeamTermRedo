# RoomieMatch — Roommate Selection App

A fully functional roommate matching web application. No backend required — runs entirely in the browser.

## Features

- **Dashboard** — Overview of match score, candidates, profile completion, and recent activity
- **Matches** — Browse and filter roommate candidates with detailed profiles, save/contact actions
- **Questionnaire** — 3-step lifestyle questionnaire with sliders, radio buttons, chips, and text input
- **Messages** — Real-time-style chat with auto-replies, conversation list, search
- **Report** — Detailed compatibility breakdown with ring chart, category scores, and insights

## Base User

Sal Rivera — Junior CS student at SUNY Oneonta. 4 candidates pre-loaded with varied compatibility scores.

## Deploy to Render

1. Push this folder to a GitHub repo
2. Go to [render.com](https://render.com) → New → Static Site
3. Connect your GitHub repo
4. Settings:
   - **Build Command:** (leave empty)
   - **Publish Directory:** `.`
5. Click Deploy

That's it — no build step, no dependencies, no server.

## Local Preview

Just open `index.html` in any browser.
